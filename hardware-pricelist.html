Click or drag a CSV file here
Accepted: .csv, .txt
📋 Expected CSV columns
Required columns:
InverterModel REQUIRED — Full model name exactly as on the box (e.g. "Sunsynk SUN-8K-SG04LP3-EU")
Brand REQUIRED — Manufacturer name exactly as registered (e.g. "Sunsynk", "Fronius") — used for NRS 097 compliance matching
ACContinuous_kW REQUIRED — Continuous AC output power in kW
DCMax_kW REQUIRED — Max DC input power in kW
UnitPrice_R REQUIRED — Price per inverter in Rands

Optional columns:
MPPT_count optional, default 2 — Number of MPPT inputs
Efficiency_pct optional, default 97 — Inverter efficiency %
InverterType optional, default Hybrid — Hybrid / Grid-Tied / Off-Grid

Price formats accepted: R18000, 18 000, 18,000.00, 18k
🔒 NRS 097-2-1 compliance is determined automatically — you do not need to enter it. The system matches your inverter (Brand + Model) against the South African Master Compliance Register. The closer your model name matches what appears on the NRS certificate, the better the match. Use the official model code where possible (e.g. "SUN-8K-SG04LP3-EU" not "8kW Hybrid").

Accepted: .csv, .txt
📋 Expected CSV columns
Required columns:
BatteryModel REQUIRED — Full model name (e.g. "Pylontech US5000")
Usable_kWh REQUIRED — Usable energy capacity in kWh
UnitPrice_R REQUIRED — Price per battery unit in Rands

Optional columns (improves calculation accuracy):
Nominal_kW optional, default 5 — Max continuous discharge power in kW
DoD_pct optional, default 85 — Max depth of discharge % — LiFePO4 = 85, lead-acid = 50
RoundtripEff_pct optional, default 92 — Round-trip charge/discharge efficiency %
Brand optional — Manufacturer name
Warranty_Years optional — Warranty period in years

Price formats accepted: R14000, 14 000, 14,000.00, 14k
Note: If DoD_pct or RoundtripEff_pct are blank, the engine applies SA spec defaults (85% / 92%).
